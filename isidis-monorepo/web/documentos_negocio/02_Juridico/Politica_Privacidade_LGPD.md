# Política de Privacidade (LGPD)

O Magicplace ("nós", "plataforma") respeita sua privacidade e protege seus dados conforme a Lei Geral de Proteção de Dados (Lei nº 13.709/2018).

1. **Coleta de Dados Pessoais:** 
   Coletamos apenas os dados estritamente essenciais para o funcionamento seguro do marketplace marketplace:
   - **Para Clientes:** Nome, Email, Histórico e logs de Pedidos e Leituras, logs de autenticação.
   - **Para Profissionais:** Nome, Email, Biografia/Foto, Chave PIX, Histórico de Saques e Transações para fins legais e contábeis.

2. **Armazenamento e Sigilo das Leituras:** 
   Os dados de leitura e a entrega de serviços interativos são confidenciais. Eles são armazenados de forma criptografada em nossos servidores providos pela Supabase e garantidos por políticas restritivas no nível do banco de dados (Row Level Security - RLS), atestando que apenas o dono do pedido tem acesso à entrega do serviço.

3. **Compartilhamento com Terceiros:** 
   Não vendemos ou cedemos dados de base de usuários para terceiros. O compartilhamento ocorre de modo automático unicamente com os provedores de pagamento homologados pela empresa (ex: Abacate Pay) para o efetivo processamento das cobranças e verificação anti-fraude.

4. **Direitos do Titular (Seus Direitos):** 
   Qualquer usuário pode solicitar ativamente a extração, edição ou exclusão permanente (anonimização sistêmica) da conta e dados vinculados através dos canais oficiais descritos no termo da plataforma. Dados de transações e logs restritos poderão ser anonimizados mas mantidos pelo tempo exigido em Lei.
