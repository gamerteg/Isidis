# Unit Economics Mensal e por Transação

Neste documento calculamos a margem real e onde estão concentrados os custos do fluxo vital de modelo de negócios atual do marketplace Magicplace.

## 1. Por Transação (Exemplo Modelagem sobre Ticket R$ 50,00)
- **Receita Bruta Gerada (TpV):** R$ 50,00  *(Cesta média / Ticket Padrão)*
- **Receita Bruta Magicplace Retida (15%):** R$ 7,50 (Fee base da plataforma)
- **Custo Pix de Acquirer (Abacate Pay):** R$ X,XX *(Depende das bases variáveis negociadas)*
- **Repasse Bruto ao Profissional (Taróloga):** R$ 42,50
- **Margem de Contribuição Bruta/Ticket:** R$ 7,50 - {Custos Acquirer Bancário} - {Custo Adicional Mensal Hospedagem Rateada}

## 2. Visão de Custos e Faturamento Fixo
- **Fluxo de Base Tecnológico Mensal:** Prover assinaturas de Cloud Services (Ex: Banco de Dados Supabase Pro Plan / Vercel Pro). Esses custos devem ser justificados por x compras mínimas globais para chegar no breakeven de base.
- **Investimentos CAC / Crescimento (Marketing Ads e Influencer):** Mensurar o quanto de mídia ou repasse é pago para que novos usuários conheçam o sistema.

*No futuro:* Planilhar métricas avançadas (CLV = Customer Lifetime Value, vs CAC) quando começar os modelos Recorrentes do Magicplace (Assinaturas Frequentes de Astrologia ou Tiragens Avulsas Mensais).
